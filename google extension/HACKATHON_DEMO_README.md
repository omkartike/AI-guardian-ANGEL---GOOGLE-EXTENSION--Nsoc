# 🛡️ Guardian AI - Hackathon Demo Edition

## 🚀 What This Extension Does

**Guardian AI** is an AI-powered security extension that analyzes websites and content using Google's Gemini AI to provide:

✅ **Website Security Analysis**
- Identifies risky websites
- Analyzes payment pages
- Detects phishing attempts

✅ **Privacy Policy & Terms Analysis**
- Summarizes long privacy policies
- Highlights concerning clauses
- Risk scoring

✅ **Content Intelligence**
- Email phishing detection
- Cookie consent management
- Real-time security warnings

---

## 🎬 Demo Steps (Perfect for Hackathon)

### Setup (Auto-Done)
1. **Extract the extension folder**
2. **Go to `chrome://extensions/`**
3. **Enable "Developer mode"** (top right)
4. **Click "Load unpacked"** and select this folder
5. ✅ Extension loads with API key pre-configured!

### Running the Demo

**Step 1: Click the Extension Icon**
- You'll see "🚀 Hackathon Edition - AI Security Scanner"
- Shows "✅ Ready for Demo!"

**Step 2: Visit Any Website**
- Privacy policies (for best results)
- Terms of service pages
- News articles
- Any regular website

**Step 3: Click "🔍 Analyze This Page"**
- Shows loading animation
- AI processes the page content
- Results appear in 2-3 seconds

**Step 4: See the Magic ✨**
- Full AI analysis of the page
- Security assessment
- Key findings
- Recommendations

## 🐛 Troubleshooting (During Demo)

| Issue | Solution |
|-------|----------|
| Extension not loading | Reload: go to chrome://extensions, toggle OFF/ON |
| "Analyze" button not working | Refresh the webpage first, then click analyze |
| No results showing | Check internet connection, wait 3 seconds |
| "Error" message | Try a different website or refresh current page |
